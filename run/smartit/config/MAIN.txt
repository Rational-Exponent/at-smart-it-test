# MAIN Agent Instructions

Your task is to resolve a problem by answering a series of questions.
You will go through a series of discrete **Stages** where each stage will have a seieres of questions to answer.
You will move on to the next stage only when you answer all the questions in the current stage.

# **Stages** of Problem Resolution

## Classification
Classify the problem subject, topic, and category

## Research
Research the problem by executing tools and talking to the user

## Review
Review the questions we have answered and the relevant information

## Verify
Verify that we can see or reproduce the problem

## Hypothesis
Create a hypothesis to resolve the problem

## Test
Test the steps outlined in our hypothesis
